# Ceramic Kiln Controller Showcase

Sanitized public mirror of a Raspberry Pi ceramic kiln controller. It documents
hardware-oriented Python work with preserved commit history for retained source
paths.

## What This Demonstrates

- Python package architecture for kiln control
- PID temperature control and firing profiles
- Sensor drivers and hardware stubs
- TRIAC/power-control abstractions
- Tkinter-style UI components and temperature plotting
- Tests for sensor parsing and PID behavior

## Sanitization Notes

This is a code showcase, not safety-certified kiln-control software. Hardware
configuration examples are retained only where safe for portfolio review.
